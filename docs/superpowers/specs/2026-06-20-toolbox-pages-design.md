# 工具箱新增页面设计规格

**日期：** 2026-06-20
**状态：** 已批准

---

## 概述

在现有肺部支气管分段标记工具的基础上，新增 2 个实用工具页面 + 1 个工具箱首页，组成一个静态工具箱。最终通过 GitHub Pages 上线。

---

## 项目结构

```
/
├── index.html                    (工具箱首页 — 新建)
├── lung-marker.html              (肺部标记 — 现有文件重命名保留)
├── tax-calc.html                 (税点计算器 — 新建)
├── rmb-upper.html                (人民币大写转换 — 新建)
├── test_toolbox.py               (整合测试 — 新建)

├── test_tax_calc.py              (税点计算器测试 — 新建，TDD)
├── test_rmb_upper.py             (大写转换测试 — 新建，TDD)
├── main.py                       (不变)
└── pyproject.toml                (不变)
```

## 设计决策：扁平文件结构

所有 HTML 文件放在根目录，不分子目录。原因：
- GitHub Pages 默认从根目录或 `/docs` 发布，扁平结构最简洁
- 三个工具页面之间通过相对路径 `./xxx.html` 互相链接
- 避免多级目录带来的 URL 嵌套问题

---

## 页面详细设计

### 1. 工具箱首页 (index.html)

**功能：** 3 个工具卡片网格，点击进入对应页面。

**布局：** 移动端优先，延续肺部标记工具的视觉风格（CSS 变量共享）。
- 顶部标题区
- 3 个卡片（2×2 网格，第三个可能跨行或居中）
- 每个卡片包含图标、标题、简介

**导航：** 各子页面顶部有 "← 返回工具箱" 链接。

### 2. 税点计算器 (tax-calc.html)

**输入：**
- 合同金额（含税总金额），输入框，支持千分位显示
- 税率：下拉选择 13% / 9% / 6% / 5% / 1% + 自定义（下拉选择"自定义"后显示输入框）

**计算逻辑：**
- 税前金额 = 含税总金额 / (1 + 税率)
- 税点金额 = 含税总金额 - 税前金额

**输出：**
- 税前金额，右侧有复制按钮
- 税点金额，右侧有复制按钮
- 复制后按钮短暂显示"已复制"反馈

### 3. 人民币大写转换器 (rmb-upper.html)

**输入：** 金额输入框

**输出：**
- 人民币大写字符串（如"壹万贰仟叁佰肆拾伍元陆角柒分"）
- 支持整数部分（亿、万、仟、佰、拾、元）和小数部分（角、分）
- 复制按钮

**转换规则：** 遵循中国人民银行《支付结算办法》中的人民币大写规范。

## 视觉风格

- 与现有肺部标记工具共享 CSS 变量体系（圆角卡片、柔和阴影、配色）
- 移动端优先 (max-width: 420px)
- 字体：PingFang SC / Noto Sans SC / Microsoft YaHei
- 交互反馈：按钮缩放、结果区淡入动画

## 测试策略 (TDD)

每个页面独立测试，Playwright：
1. **工具箱首页：** 3 个卡片渲染、点击跳转正确
2. **税点计算器：** 金额输入 → 税率选择 → 计算正确性 → 复制按钮功能
3. **大写转换器：** 金额输入 → 大写结果正确性 → 复制按钮功能
4. **跨页面导航：** 首页 → 子页面 → 返回首页，完整流程
